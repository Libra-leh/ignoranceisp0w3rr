THE LIBRALEH
A website that helps you track your readings, allow you to give ratings,reviews,share with friends.
Feature: 
+ Work in both English and Vietnamese.
+ Add book to your library using integrated search bar,also you can upload your own bookcover as well
+ You can log in your account using any device as long as you remember your username and password since they are all stored in a database
+ Allow you to give ratings, reviews and share with friend. By copying a link, others get to see your library in read-only mode
+ Friend section is there for you.Copy friends' read-only link to get to their library anytime.
+ You can add any book that you find interesting in others' library.Bookcover,name,author is store in localstorage and paste to your library once you're loggin
+ Daily knowledge section offers you trivial understanding everyday.
+ Real-time updating across all site
+ Personalize your library with adding categories of your own feature.
   How it work:
  +English bookcover is supported by Google Books API, while Vietnamese bookcover is found by scraping Tiki's api through a Cloundfare worker account
  +Data is store temporarily in localstorage and permanently in Firebase.
  +Frontend: htlm, JavaScript, CCS 
  +Hosted by GitHub
